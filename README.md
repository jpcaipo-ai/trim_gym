# Trim Gym Dashboard

Tablero local de control comercial para Trim Gym y Llama Leads.

## Archivo principal

Abrir en navegador:

`outputs/control_dashboard/tablero_control_trim_gym.html`

## Regenerar tablero

```powershell
node build_control_data.mjs
node build_all_sedes_metrics.mjs
node build_control_dashboard_html.mjs
```
